# IF Achados

Sistema web de Achados e Perdidos desenvolvido para o Instituto Federal do Maranhão (IFMA). O projeto permite cadastrar, consultar, editar e gerenciar objetos encontrados na instituição, facilitando sua identificação e devolução aos proprietários.

## Objetivo

O IF-Achados foi desenvolvido com o objetivo de informatizar o processo de registro e gerenciamento de objetos perdidos e encontrados dentro do IFMA, proporcionando uma plataforma simples, organizada e segura para alunos, professores e servidores.

- Java
- MySQL
- Vite
- Git
- GitHub

## Estrutura do projeto

- `backend/`: pasta destinada ao backend Spring Boot em Java.
- `frontend/`: aplicação frontend React/Vite.
- `banco/`: scripts SQL, modelo do banco e DER.
- `docs/`: documentação técnica, relatórios e diagramas.

## Como executar

Backend

Acesse a pasta `backend/` e execute o projeto Spring Boot com o Maven Wrapper quando o backend estiver disponível.

Frontend

Acesse a pasta `frontend/` e utilize os scripts definidos em `package.json`. cd frontend depois npm run dev

## Segurança

O sistema utiliza:

- Spring Security
- JWT (JSON Web Token)
- Rotas públicas e privadas
- Autenticação via Bearer Token

## Principais Recursos

- Cadastro de usuários
- Login seguro
- Cadastro de objetos
- Upload de imagens
- Busca de objetos
- Visualização de detalhes
- Edição de registros
- Exclusão de registros
- Painel administrativo
- Integração completa com banco de dados
