# IF Localiza

Sistema de achados e perdidos preparado para evoluir com API, framework, banco de dados e bibliotecas.

## Estrutura

- `index.html`: página principal do sistema.
- `assets/css`: estilos da interface.
- `assets/js`: scripts do front-end.
- `assets/images`: imagens usadas no projeto.
- `assets/icons`: ícones e favicons.
- `assets/fonts`: fontes locais.
- `src/api`: integrações com APIs e rotas futuras.
- `src/components`: componentes reutilizáveis da interface.
- `src/config`: configurações do projeto.
- `src/database`: conexão, migrations, seeds e modelos de banco.
- `src/layouts`: estruturas de páginas e layouts.
- `src/lib`: bibliotecas internas e adaptadores.
- `src/pages`: páginas ou views da aplicação.
- `src/services`: regras de negócio e comunicação entre camadas.
- `src/utils`: funções auxiliares.
- `docs`: documentação e anotações do projeto.
- `tests`: testes automatizados.

## Observações

- O CSS e o JavaScript foram apenas movidos para suas pastas próprias.
- O HTML já possui pontos de integração com `data-*`, formulário de cadastro, busca e template para renderização dinâmica de itens.
